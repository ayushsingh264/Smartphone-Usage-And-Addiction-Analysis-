and Machine Learning.

- Pandas
- Seaborn
- Scikit-learn
- Power BI

- Data Cleaning
- EDA
- Dashboard
- Machine Learning Prediction
  
# Results
- Identified addiction patterns
- Predicted addiction levels
- Built interactive dashboard
